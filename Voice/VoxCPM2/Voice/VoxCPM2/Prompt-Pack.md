## 1. Инсталация и първи старт

**Кога да го ползваш:** Настройваш средата от нулата и искаш да провериш дали всичко е наред преди да пишеш собствен код.

```

Средата ми: Python 3.11, PyTorch 2.5.0, CUDA 12.0, GPU NVIDIA RTX 4090.
Стъпките трябва да включват:
1. pip install командата
2. Зареждане на модела с VoxCPM.from_pretrained('openbmb/VoxCPM2')
3. Генериране на кратко тестово изречение на английски
4. Запис на изхода като .wav файл

Покажи целия код, не съкращавай.

```

```
Напиши Python код, който подава следния смесен текст към модела и го синтезира в един аудио файл:

Текст: "Hello, this is a test. Bonjour, c'est un essai. Здравейте, това е тест."

Не задавай езиков параметър. Запиши резултата на 48kHz. Обясни защо VoxCPM2 не изисква ръчна езикова анотация.
```
