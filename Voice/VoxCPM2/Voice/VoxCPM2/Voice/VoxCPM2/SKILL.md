# VoxCPM2 — скил за синтез на реч

## Какво прави моделът

VoxCPM2 е TTS система с 2B параметъра, изградена върху MiniCPM-4. Генерира реч директно — без дискретна токенизация — чрез дифузионно-авторегресивна архитектура (end-to-end). Изходът е 48kHz аудио с вградена супер-резолюция чрез asymmetric AudioVAE V2.

Поддържа 30 езика без да изискваш езиков таг. Обучен върху над 2 милиона часа многоезична реч.

## Инсталация

Изисква Python 3.10–3.12, PyTorch 2.5.0+ и CUDA 12.0+.

```bash
pip install voxcpm
```

## Основна употреба (Python API)

```python
from voxcpm import VoxCPM

model = VoxCPM.from_pretrained('openbmb/VoxCPM2')
```

От тук нататък избираш режим според нуждата.

## Режими на работа

### Voice Design — глас от текстово описание

Създаваш нов глас само с естествен текст. Без референтен клип, без ръчна настройка.

```python
# Примерен параметър — провери документацията за точния API
model.design_voice(description="...")
```

### Controllable Cloning — клониране с насока

Клонираш глас от кратък референтен клип. По желание добавяш стилови указания, за да контролираш интонация или темпо.

### Ultimate Cloning — пълно клониране

Възпроизвежда всеки нюанс на оригинала — тембър, ритъм, емоция, стил. Ползваш, когато точността е критична.

## Производителност в реално време

На NVIDIA RTX 4090:

- RTF ~0.3 в стандартен режим
- RTF ~0.13 с Nano-vLLM / vLLM-Omni

Стрийминг е наличен и в двата режима.
